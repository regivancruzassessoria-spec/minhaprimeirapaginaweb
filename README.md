# 🌐 Meus Sites Favoritos
Projeto desenvolvido para a disciplina de **Programação Web I** do curso Técnico em Desenvolvimento de Sistemas.

## 📚 Sobre o Projeto
Esta página web foi criada utilizando os conceitos básicos de HTML5
O objetivo da atividade é apresentar uma lista com meus sites favoritos, utilizando:

- Estrutura HTML5
- Títulos
- Imagens
- Links
- Figuras e legendas
- Comentários no código
- Organização visual com CSS

## 🖥️ Tecnologias Utilizadas

- HTML5
- CSS3
- Visual Studio Code
## 📌 Sites Escolhidos
Escolhi os sites abaixo por serem aqueles que navego quase que diariamente.
- Jusbrasil
- OAB SP
- Escavador
- Trivago
- Aprenda Mais
